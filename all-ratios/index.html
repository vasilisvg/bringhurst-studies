There's an overview of all ratios that can be created with the square, the pentagon, the hexagon and the octagon on page 148 and 149 in The Elements of Typographic Scale. All these ratios have a name. I was wondering what they look like, so I created this page. If you resize this page you'll see the name of the ratio of the rectangle you're looking at. If you open this page on a device you can not resize you'll see the name of the ratio that browser uses. Some ratios are considered to be more boring than others. Can you see which ones?
